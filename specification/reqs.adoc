= Requirements

* The entire application must be accessible on the open internet.
* The public-facing home page must be usable on both desktop and mobile.
* The public-facing home page must show the following: a page title, a "provoke me" button, social media share buttons, a "powered by Namahn" link, a "find out more" link, and a feedback button.
* The "provoke me" button shows a different message each time, until there are no more messages to show.
* The management interface must be password-protected.
